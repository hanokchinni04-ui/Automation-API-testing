# AutomationExercise API Testing with Postman

## Description
API Testing for automationexercise.com using Postman

## Tools Used
- Postman
- JavaScript (Test Scripts)

## APIs Covered
- API 1 - Get All Products
- API 2 - POST To Products List
- API 3 - Get All Brands List
- API 4 - PUT To Brands List
- API 5 - Search Product
- API 6 - Search Product Without Parameter
- API 7 - Verify Login Valid
- API 8 - Verify Login Without Email
- API 9 - DELETE To Verify Login
- API 10 - Verify Login Invalid Details
- API 11 - Create User Account
- API 12 - Delete User Account
- API 13 - Update User Account
- API 14 - Get User Detail By Email

## Test Coverage
- Positive Testing
- Negative Testing
- CRUD Operations
- Environment Variables
- Test Scripts (Assertions)
- Collection Runner

## How to Run
1. Import collection in Postman
2. Set up Environment Variables
3. Run Collection Runner
